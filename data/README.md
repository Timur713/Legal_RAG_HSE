# Data Layout

`raw/` хранит исходные файлы конкурса и не должен коммититься в Git.

Ожидаемые файлы в `raw/`:

- `documents.csv`
- `train.csv`
- `test.csv`
- `sample_submission.csv`

`processed/` предназначен для производных датасетов и промежуточной предобработки. Тяжелые артефакты лучше не коммитить.
